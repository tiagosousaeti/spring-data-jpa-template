# Spring Data JPA Template

Projeto Java 8 com Spring Data Jpa

### Clonando o projeto

Requer o Git configurado. No terminal, use o comando: 

git clone https://github.com/tiagosousaeti/spring-data-jpa-template.git

## Dependências
- Java 8
- Log4j 2.10.0
- Maven Compiler 3.2
- Servlet API 3.1.0
- Spring MVC 5.0.2
- Thymeleaf 3.0.9
